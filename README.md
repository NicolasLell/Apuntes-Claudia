#Apuntes Claudia 📝
Este es un proyecto simple de pruebas, creado con el fin de experimentar y aprender a integrar herramientas de inteligencia artificial. 

¿Qué hace?
Organización: Permite elegir la materia y el día de la cursada.

Registro: Espacio para agregar notas y resúmenes de las clases.

Planificación: Genera un Roadmap de estudio basado en los apuntes para saber por dónde empezar a estudiar.
